## 关键公式速查

| 量 | 公式 |
|---|---|
| 终端速度 | $v_g = L / \bar{t}$ |
| 未修正半径 | $r = \sqrt{9\eta v_g / [2g(\rho_{\text{油}}-\rho_{\text{空}})]}$ |
| 康宁汉修正半径 | $r' = -b_0/(2p) + \sqrt{(b_0/(2p))^2 + r^2}$ |
| 修正质量 | $m = \frac{4}{3}\pi r'^3 (\rho_{\text{油}}-\rho_{\text{空}})$ |
| 电荷量 | $q = mgd/U$ |
| 基本电荷 | $e = q/n$，$n$ 由公约数法确定 |
